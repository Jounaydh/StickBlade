# StickBlade

Lightweight 3D third-person action prototype for Unity.

## Open The Project

1. Install Unity 6.3 LTS, or Unity 6.0 LTS if 6.3 is not available.
2. Open this folder in Unity Hub.
3. Let Unity import packages.
4. In Unity, run:

   `Tools > StickBlade > Build Prototype Arena`

5. Press Play.

## Prototype Controls

- Move: `WASD`
- Lock on: `Tab`
- Attack: left mouse button
- Parry: right mouse button

## Current Scope

This is intentionally small:

- Third-person movement
- Camera follow
- Lock-on targeting
- Sword attack
- Basic enemy
- Health bars
- Posture bars
- Parry/deflect
- Deathblow when enemy posture breaks
- Simple arena and placeholder shapes

No multiplayer, inventory, quests, open world, realistic graphics, or RPG systems yet.
